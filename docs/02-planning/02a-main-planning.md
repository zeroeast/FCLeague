## 3. 시즌 구조

- 1시즌 = 8명 감독, 상대당 2경기 = **총 56경기**
- 경기 결과는 **참여 감독이 직접 수기 입력**
- 시즌 기간 및 활성 여부는 관리자가 설정
- 포인트는 **매 시즌 초기화 없이 누적** (약 5~6시즌 후 운영자 수동 초기화)

## 6. 감독 특성 시스템

8명 감독이 공유하는 **특성 풀 7종**에서 감독마다 2~4개가 이름 시드(seed) 기반으로 랜덤 배정된다.

### 6-1. 특성 목록

| ID | 이름 | 타입 | 비용 | 효과 |
|----|------|------|------|------|
| `ban-player` | 선수 밴 능력 | 소모 아이템 | 10P | 상대 선수 1명 출전 불가 (경기당 1회) |
| `reduce-squad` | 선수단 감소 능력 | 소모 아이템 | 20P | 상대 선수단 등록 인원 10명으로 제한 |
| `training-coach` | 능숙한 훈련감독 | 감독 특성 | — | n명(2~5) 선수에게 훈련 코치 부착 |
| `sub-master` | 교체의 명장 | 감독 특성 | — | 교체 선수 (n BP, 200~500) 규모 운용 |
| `doctor-strange` | 닥터스트레인지 | 소모 아이템 | 10P | 선수 1명 재선발 (경기당 1회) |
| `enhance-master` | 강화의 달인 | 감독 특성 | — | 강화 성공률 n%(3~8%) 상승 |
| `draw-master` | 뽑기의 달인 | 감독 특성 | — | n%(8~20%) 확률로 재뽑기 활성화 |

- **소모 아이템** (`oneShot: true`): 사용 시 포인트 소모, 해당 경기 1회 적용
- **감독 특성** (`oneShot: false`): 지속 효과, 포인트 소모 없음

### 6-2. 구현 파일

- 특성 데이터: `frontend/src/constants/managerTraits.js`
- 상점 가격: `frontend/src/constants/shopProducts.js`
  - 소모 아이템(active): 60P
  - 감독 특성(passive): 120P

### 6-3. 감독 특성 동적 값

감독 특성의 수치 값은 감독 이름 seed로 결정된다.

| 특성 | 범위 |
|------|------|
| 훈련감독 코치 수 | seed % 4 + 2 → 2~5명 |
| 교체 BP 한도 | (200 + seed%7×50) → 200~500 BP |
| 강화 성공률 보너스 | seed % 6 + 3 → 3~8% |
| 재뽑기 확률 | seed % 13 + 8 → 8~20% |

## 7. 선수 카드 UI 규칙

구현 파일: `frontend/src/constants/playerColors.js`

### 7-1. 강화 단계 색상

| 강화 등급 | 색상 |
|----------|------|
| +0 ~ +4 | 회색 (`#94a3b8`) |
| +5 ~ +7 | 보라 (`#8b5cf6`) |
| +8 ~ +10 | 금색 (`#FFD700`) + 강한 글로우 |

### 7-2. 포지션 색상

| 그룹 | 포지션 | 색상 |
|------|--------|------|
| GK | GK | 주황 (`#f97316`) |
| 수비 | CB, LB, RB, LWB, RWB | 파랑 (`#3b82f6`) |
| 미드 | CM, CAM, CDM, LM, RM | 녹색 (`#22c55e`) |
| 공격 | ST, LW, RW, CF | 빨강 (`#ef4444`) |

### 7-3. OVR 티어

OVR 구간별 선수 이름색·슬롯 배경·테두리 자동 적용 (97+, 94+, 91+, 88+, 85+, 82+, 그 외).
