# 패널 기획서 — 승부 예측

위치: `/shop` 페이지 내 4번째 탭 (기존 뽑기 / 강화 / 특성 우측)  
탭명: `승부 예측`  
컴포넌트 예정: `pages/Shop.jsx` 내 `PredictionTab.jsx`  
구현 상태: 미구현

---

## 목적

실제 축구 경기 결과를 포인트로 예측하여, 적중 시 배당 기반 포인트를 획득하는 콘텐츠.
포인트 소모처 중 유일한 **증식형** 콘텐츠.

## 이벤트 카드 상태

| 상태 | 표시 |
|------|------|
| `open` | 배당 표시, 예측 참여 가능 |
| `closed` | 마감됨, 참여 불가, 배당 고정 표시 |
| `settled` | 결과 확정, 적중·실패 표시, 지급 포인트 표시 |
| `cancelled` | 취소됨, 참여 포인트 환불 표시 |

---

## 예측 참여 플로우

```
이벤트 카드에서 결과 선택 (한국 승 / 무승부 / 일본 승)
    ↓
포인트 입력 모달
  - 참여할 포인트 입력 (최소 50P)
  - 예상 획득 포인트 미리보기: 입력P × 확정 배당
  - 현재 잔액 표시
    ↓
[확인] 버튼 → 서버 POST → 잔액 차감 + 예측 기록 생성
    ↓
카드에 "내 예측: OO 승 · NP 참여중" 표시
```

---

## 포인트 입력 모달 상세

| 항목 | 내용 |
|------|------|
| 선택한 결과 | 예: "한국 승" |
| 확정 배당 | 예: 2.10 |
| 참여 포인트 입력 | 숫자 입력 (최소 50P, 잔액 초과 불가) |
| 예상 획득 | `floor(입력P × 배당)` 실시간 표시 |
| 현재 잔액 | 예: 1,200P |
| [확인] / [취소] | |

---

## 마감 후 상태

- 마감 이후 카드는 **잠금 상태**로 표시
- 배당 고정 뱃지 표시: "배당 확정 · 마감됨"
- 참여했다면 "내 예측: OO · NP 참여중" 유지

---

## 결과 정산 후 상태

적중 시:
```
✅ 한국 승 적중!
참여: 100P  →  획득: 210P  (+110P)
```

실패 시:
```
❌ 예측 실패
참여: 100P  →  소멸
```

## 데이터 요구사항

| 데이터 | API |
|--------|-----|
| 진행 중 이벤트 목록 | `GET /api/predictions/events?status=open` |
| 종료된 이벤트 목록 | `GET /api/predictions/events?status=settled&limit=5` |
| 내 예측 현황 | `GET /api/predictions/my` |
| 예측 참여 | `POST /api/predictions/stake` |
| 포인트 잔액 | `GET /api/wallet/me` |

---

## 운영자 관련 (관리 페이지)

| 기능 | 설명 |
|------|------|
| 이벤트 등록 | 경기명, 옵션(결과 선택지), 배당, 마감 시각 입력 |
| 배당 동기화 | 외부 배당 API 참고값 불러오기 |
| 예측 마감 | 수동 마감 버튼 (자동 마감은 마감 시각 기준) |
| 결과 입력 | 실제 경기 결과 선택 |
| 정산 실행 | 결과 기반 일괄 정산 (중복 방지 처리) |
| 이벤트 취소 | 경기 취소 시 전액 환불 처리 |

---

## 계산식

```
예상 획득 포인트 = floor(참여P × 확정배당)
예: 100P × 2.10 = 210P
예: 150P × 3.30 = 495P
```
