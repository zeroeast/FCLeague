# 패널 기획서 — 통계

경로: `/stats`  
컴포넌트: `pages/Stats.jsx`  
구현 상태: 샘플 UI 완료 (API 연동 필요)

---

## 목적

팀·선수 성적 대시보드. "누가 제일 잘하는가", "어떤 선수가 활약했는가"를 한눈에 보여준다.

---

## 레이아웃 구조

```
[탭 영역: 팀 | 선수]

[메인 대형 카드 (좌)]        [요약 카드 (우)]

[하단 랭킹 카드 1]  [하단 랭킹 카드 2]  [하단 랭킹 카드 3]
```

---

## 팀 탭

| 카드 | 제목 | 내용 |
|------|------|------|
| 메인 (좌) | 리그 종합 순위 | 전적 순위표. 순위·구단주·경기수·승/무/패·승률·득점·실점·골득실·최근5경기(W/D/L 배지) |
| 요약 (우) | 현재 1위 팀 | 1위 감독 요약. 승률·총경기·골득실·최근폼·코멘트 |
| 하단 1 | 팀 총 득점 순위 | 구단주별 총 득점 랭킹. 순위·구단주·총득점·경기당득점 |
| 하단 2 | 팀 골득실 순위 | 구단주별 골득실 랭킹. 득점·실점·골득실 |
| 하단 3 | 최소 실점 순위 | 구단주별 최소 실점 랭킹. 총실점·경기당실점 |

---

## 선수 탭

| 카드 | 제목 | 내용 |
|------|------|------|
| 메인 (좌) | 현재 선수 파워랭킹 | TOP 10. 순위·선수명·시즌·포지션·출전수·골·도움·파워점수 |
| 요약 (우) | 워스트 퍼포먼스 | 기대 대비 부진 선수. 선수명·시즌·출전수·골·도움·부진 사유 |
| 하단 1 | 득점 순위 | 선수별 총 득점 랭킹 |
| 하단 2 | 어시스트 순위 | 선수별 도움 랭킹 |
| 하단 3 | 수비 순위 | 수비 기여도 랭킹 |

### 파워점수 계산 (선수 탭)

```
파워점수 = (골 × 3) + (도움 × 2) + (출전수 × 0.5)
```
> 확정 공식 아님. 백엔드 구현 시 조정 가능.

### 수비점수 계산

```
수비점수 = 태클 + 차단 + 클리어 + 수비포지션 기여도
```
> 현재 `match_stats`에 수비 기록 컬럼 미존재. 스키마 확장 필요.

## 공통 UI 규칙

- 1~3위: 금/은/동 배지
- 각 카드 우상단: 기간 필터 (`전체 기간 / 최근 10경기 / 이번 달`)
- 랭킹 항목 클릭 → 해당 유저·선수 상세 페이지 이동 (미구현)
- 데이터 없을 때: "아직 수집된 경기 데이터가 없습니다" 빈 상태 메시지

## 데이터 요구사항

| 데이터 | API |
|--------|-----|
| 팀 종합 순위 | `GET /api/standings?season_id=` |
| 팀 득점·실점 랭킹 | `GET /api/stats/teams?season_id=` |
| 선수 파워랭킹 | `GET /api/stats/players?season_id=` |
| 득점·어시스트 랭킹 | `GET /api/stats/scorers?season_id=` |
