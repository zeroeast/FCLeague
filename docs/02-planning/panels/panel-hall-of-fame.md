## 목적

시즌별 리그 우승·컵 우승·개인 수상 기록을 보관하는 화려한 아카이브 페이지.

---

## 화면 구성

### 상단 시즌 탭

- 시즌별로 탭 전환
- 예: 시즌 0 / 시즌 1 / 시즌 2 ...

### 챔피언 카드 (시즌 메인)

- 시즌 우승 감독 이름·엠블럼
- 우승 AI 이미지 (Supabase Storage)
- 글로우 애니메이션 효과

### 수상 기록 카드

| 항목 | 내용 |
|------|------|
| 우승팀 | 시즌 우승 감독 |
| 득점왕 | 시즌 최다 득점 선수·감독 |
| 도움왕 | 시즌 최다 도움 선수·감독 |
| MVP | 시즌 최우수 선수·감독 |
| 추가 항목 | 운영자가 자유롭게 추가 가능 |
