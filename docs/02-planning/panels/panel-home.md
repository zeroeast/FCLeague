## 목적

리그 현황을 한눈에 보여주는 대시보드. 접속 시 가장 먼저 보이는 화면.

---

## 화면 구성

### 상단 헤더 배너
- 현재 활성 시즌 이름 + LIVE 배지
- 참가 감독 수 / 진행 경기 수 / 총 득점 수
- 1위 감독 강조 표시

### 우승 테마 배경 (미구현)
- 최근 시즌 우승 감독/팀 컨셉에 맞는 배너·배경 적용
- 우승 감독의 엠블럼 컬러를 배경에 반영

### 메인 콘텐츠 영역

| 섹션 | 내용 |
|------|------|
| 감독 순위표 | EPL 스타일. 순위·엠블럼·감독명·경기수·승/무/패·득실차·승점 |
| 최근 경기 결과 | 최근 완료된 경기 3~5개. 홈/어웨이·스코어 |
| 득점 순위 TOP 5 | 득점 많은 감독 5명. 이름·득점 |
| 다음 경기 | 예정된 다음 경기 1건 표시 |

---

## 데이터 요구사항

| 데이터 | 출처 | API |
|--------|------|-----|
| 현재 활성 시즌 | `seasons` | `GET /api/seasons/active` |
| 리그 순위표 | `matches` 집계 | `GET /api/standings?season_id=` |
| 최근 경기 | `matches` | `GET /api/matches?status=played&limit=5` |
| 득점 순위 | `match_stats` 집계 | `GET /api/stats/top-scorers` |
