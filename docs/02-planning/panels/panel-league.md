# 패널 기획서 — 리그

경로: `/league`  
서브 탭: `/league/schedule` (경기 일정) / `/league/standings` (순위표)  
컴포넌트: `pages/LeagueLayout.jsx`, `LeagueSchedule.jsx`, `LeagueStandings.jsx`  
구현 상태: 샘플 UI 완료 (결과 입력·API 연동 필요)

---

## 목적

경기 일정 확인, 결과 입력, 순위표 조회.

---

## 서브 탭 구성

### 일정 탭 (`/league/schedule`)

- 전체 경기 목록 (날짜별 그룹)
- 완료된 경기: 홈/어웨이 팀명 + 스코어
- 예정된 경기: 홈/어웨이 팀명 + 날짜
- **경기 결과 입력 버튼** (본인 경기에만 활성화, 로그인 필요)

### 순위표 탭 (`/league/standings`)

| 컬럼 | 내용 |
|------|------|
| 순위 | 1~8 (1~3위 강조) |
| 엠블럼 | SVG 방패 |
| 감독명 | |
| 경기 수 | |
| 승 / 무 / 패 | |
| 득점 / 실점 / 골득실 | |
| 승점 | |

---

## 경기 결과 입력 흐름

```
감독이 경기 결과 입력 버튼 클릭
    ↓
홈/어웨이 스코어 입력 모달
    ↓
제출 → 서버 저장 → 순위표 자동 갱신
```

- 입력 가능 조건: 해당 경기의 홈 또는 어웨이 감독 본인
- 결과 수정: 관리자만 가능

---

## 시즌 구조

- 8명 감독, 상대당 2경기 = **총 56경기 / 시즌**
- 승점: 승=3, 무=1, 패=0
- 동점 시 순위 결정: 골득실 → 득점 → 직접 대결 결과

---

## 데이터 요구사항

| 데이터 | API |
|--------|-----|
| 시즌 경기 목록 | `GET /api/matches?season_id=` |
| 순위표 | `GET /api/standings?season_id=` |
| 경기 결과 입력 | `PATCH /api/matches/:id/result` |
