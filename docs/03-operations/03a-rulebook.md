# 운용 규정집

> FC온 리그사이트의 공식 운영 규정.
> 기획서를 기반으로 한 구체적인 운영 기준이며, 사이트 내 공지·안내 페이지로도 활용 가능한 내용이다.

---

## 1. 총칙

### 1-1. 서비스 성격

FC온 리그사이트는 FC온라인을 즐기는 지인들 간의 자체 리그를 기록·운영하는 **폐쇄형 커뮤니티 서비스**다.
외부 공개를 전제로 하지 않으며, 초대된 감독만 참여할 수 있다.

### 1-2. 포인트 성격

사이트 내 포인트는 **비현금성 재화**다.

- 현금 충전 불가
- 현금 환전 불가
- 상품·기프티콘 교환 불가
- 외부 플랫폼 이전 불가

포인트는 사이트 내 콘텐츠 이용·랭킹·기록·명예 요소를 위해서만 사용된다.

---

## 2. 리그 운영 규정

### 2-1. 시즌 구조

- 1시즌 = 8명 감독, 상대방과 각 2경기 = **총 56경기**
- 시즌 시작·종료일은 운영자가 결정
- 동시에 활성화된 시즌은 1개만 허용

### 2-2. 경기 결과 입력

- 경기 결과는 **해당 경기의 참여 감독(홈 또는 어웨이)이 직접 입력**한다
- 결과 입력 가능 기간: 경기 예정일로부터 **7일 이내**
- 결과 수정·삭제: **관리자만** 가능
- 허위 결과 입력 시 운영자 판단 하에 취소 처리

### 2-3. 순위 결정 방식

```
기본: 승점 (승=3, 무=1, 패=0)
동점 시 순서:
  1. 골득실 (득점 - 실점)
  2. 총 득점 수
  3. 직접 대결 결과
  4. 운영자 재경기 결정
```

### 2-4. 시즌 종료 처리

시즌 종료 시 운영자가 다음을 수행한다.

1. 최종 순위 확정
2. 시즌 순위 보상 포인트 지급
3. 명예의 전당 기록 등록
4. 우승 감독 엠블럼에 ★ 추가
5. 다음 시즌 개설 공지

---

## 3. 포인트 운영 규정

### 3-1. 포인트 지급 기준

| 지급 사유 | 지급량 (기준값, 시즌별 조정 가능) |
|----------|-------------------------------|
| 리그 경기 승리 | +30P |
| 리그 경기 무승부 | +10P |
| 리그 경기 패배 | +0P |
| 시즌 종료 1위 | +1000P |
| 시즌 종료 2위 | +700P |
| 시즌 종료 3위 | +500P |
| 시즌 종료 4위 | +300P |
| 시즌 참가 보상 | +100P |

> 위 수치는 초기 설정값. 테스트 시즌 데이터 수집 후 조정.

### 3-2. 포인트 소모 기준

| 소모 항목 | 비용 (기준값) |
|----------|--------------|
| 선수 뽑기 1회 | 700P |
| 선수 강화 1회 (단계별 차등) | 100P ~ 2200P |
| 소모형 특성 구매 | 60P |
| 패시브 특성 구매 | 120P |
| 승부 예측 1회 최소 참여 | 50P |

> 재화 밸런싱 기획서(`02b-currency-system.md`) §7 기준으로 조정.

### 3-3. 포인트 투명성

- 모든 포인트 변동 내역은 사이트에서 **전체 공개**한다
- 유저는 자신 및 타인의 포인트 획득·소모 내역을 조회할 수 있다
- 운영자가 지급·회수한 내역도 공개 히스토리에 포함된다

### 3-4. 포인트 초기화

- 포인트는 매 시즌 초기화하지 않으며, **여러 시즌에 걸쳐 누적**된다
- 전체 초기화는 운영자가 약 **5~6시즌 이후** 별도 공지 후 수행한다
- 초기화 시점에는 누적 자산에 비례한 **별도 명예 보상**을 지급한다
- 초기화 3시즌 전부터 예고 공지를 게시한다

## 4. 승부 예측 운영 규정

### 4-1. 예측 대상 경기

- 운영자가 지정한 실제 축구 경기 (월드컵, 챔피언스리그, 국가대표, FC 이벤트 등)
- 예측 대상은 결과가 명확한 공식 경기에 한한다

### 4-2. 배당 정책

- 외부 스포츠 배당 API를 참고한 **고정 배당** 방식 사용
- 배당은 **예측 마감 시점에 고정**되며, 이후 변경 불가
- 배당 설정 권한: 운영자만

### 4-3. 예측 마감

- 경기 시작 시점 또는 운영자가 지정한 시각에 마감
- 마감 이후: 신규 예측 참여 불가 / 기존 예측 수정·취소 불가 / 배당 변경 불가

### 4-4. 결과 정산

- 경기 종료 후 결과를 자동 동기화하는 것을 기본으로 한다
- 오류·API 지연 시 운영자 수동 동기화 가능
- **중복 정산 불가** — 동일 경기·유저 예측에 보상은 1회만 지급
- 정산 오류 발생 시 운영자가 `CORRECTION` 트랜잭션으로 수동 보정

### 4-5. 예측 취소·환불

- 마감 전: 운영자 판단 하에 취소 가능 (경기 취소·연기 등 불가항력)
- 마감 후: 취소 불가 (경기 취소 시 운영자가 전액 환불 처리)

---

## 5. 선수 뽑기·강화 운영 규정

### 5-1. 선수 뽑기

- 획득한 선수는 해당 감독의 선수단에 자동 추가
- 뽑기 결과는 취소·환불 불가
- 뽑기 확률표는 사이트 내 공개

### 5-2. 선수 강화

- 강화 실패 시 포인트는 소모되며, 선수는 현재 등급을 유지한다 (하락 없음)
- 강화 성공·실패 결과는 모두 기록에 남는다
- 강화 비용·성공률은 운영자가 시즌 단위로 조정 가능

### 5-3. 특성 구매

- 구매한 특성은 해당 감독에게 영구 귀속
- 소모형 특성은 1회 사용 후 소멸하며, 재구매 가능
- 패시브 특성은 보유 기간 동안 지속 효과 적용

---

## 6. 관리자 권한 규정

### 6-1. 관리자 역할 (`role: 'admin'`)

관리자는 다음을 수행할 수 있다.

- 시즌 생성·종료·설정 변경
- 경기 일정 등록·수정
- 경기 결과 수정·삭제
- 포인트 지급·회수 (원장 기록 필수)
- 배당 설정·결과 동기화·예측 정산
- 상품 가격·강화 확률 조정
- 전체 포인트 초기화 (공지 후)
- 명예의 전당 기록 등록·이미지 업로드
- 유저 포인트 내역 전체 조회

### 6-2. 관리자 제한

- 관리자도 포인트 잔액을 직접 수정할 수 없다. 반드시 원장 트랜잭션으로 처리.
- 관리자의 포인트 지급·회수 내역도 공개 히스토리에 표시된다.

---

## 7. 개인정보 및 계정 규정

- 닉네임과 비밀번호로 로그인 (Google OAuth 없음)
- 비밀번호는 bcryptjs로 해싱 저장 (평문 저장 절대 금지)
- 계정 삭제 요청 시 운영자가 처리
- 넥슨 계정 연동은 선택 사항 (`nexon_ouid`)
