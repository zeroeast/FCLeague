# FC온라인 리그 사이트 기획서

## 1. 프로젝트 개요

FC온라인을 즐기는 10명 내외 친구들을 위한 자체 리그 관리 웹사이트.
경기 결과 기록, 팀 관리, 선수 검색, 통계, 명예의 전당 등을 제공한다.

---

## 2. 참여 감독

| 감독명 | 비고 |
|--------|------|
| 준현   |      |
| 종성   |      |
| 영동   |      |
| 민혁   |      |
| 삼주   |      |
| 영모   |      |
| 진수   |      |
| 기성   |      |

- 각 감독은 고유한 **앰블럼(이미지)** 을 보유
- 앰블럼은 Supabase Storage에 저장, DB에는 URL만 보관
- 우승 횟수에 따라 앰블럼 하단에 **별(★)** 이 추가됨

---

## 3. 시즌 구조

- 한 시즌당 각 감독은 나머지 7명과 **2판씩** 대결 (총 56경기/시즌)
- 경기 결과는 **참여 감독이 직접 수기 입력**
- 시즌 기간 및 활성 여부는 관리자가 설정

## 4. 내비게이션 메뉴

> **팀 = 감독** — 1감독 1팀 개념으로 통일

```
홈  /  리그  /  팀  /  선수 검색  /  통계  /  명예의 전당  /  상점
```

| 메뉴 | 경로 | 하위 메뉴 | 구현 상태 |
|------|------|-----------|-----------|
| 홈 | `/` | — | 샘플 UI 완료 |
| 리그 | `/league` | 경기 일정 `/league/schedule` · 순위표 `/league/standings` | 샘플 UI 완료 |
| 팀 | `/teams` | 내 팀 정보 `/teams/my` · 전체 팀 정보 `/teams/all` | 샘플 UI 완료 |
| 선수 검색 | `/players` | — | UI 골격만 |
| 통계 | `/stats` | — | 샘플 UI 완료 |
| 명예의 전당 | `/hall-of-fame` | — | 샘플 UI 완료 |
| 상점 | `/shop` | 패시브 스킬 · 액티브 스킬 | 샘플 UI 완료 |
| 로그인 | `/login` | — | UI 골격만 |

**제거된 메뉴** (레거시 경로는 `/shop`으로 리다이렉트)
- ~~특성 투자~~ (`/traits`) → 상점으로 통합
- ~~선수 뽑기~~ (`/gacha`) → 메뉴에서 제거

> 백엔드/API 연동 전 단계이며, 현재는 프론트엔드 샘플 데이터 기반으로 동작한다.

## 5. 페이지별 기획

### 5-1. 홈 (대시보드)

**기획**
- **감독 순위표** — EPL 순위표 스타일 (승점, 승/무/패, 득실차)
- **통계 대시보드** — 감독 득점 순위, 선수 득점 순위, 득점력/수비력 지표
- **이번 시즌 정보** — 리그 기간, 활성 시즌 표시
- **우승 테마 배경** — 최근 시즌 우승 감독/팀 컨셉에 맞는 배너/배경 적용

**구현됨 (샘플 UI)**
- 리그 현황판 (LIVE 배지, 참가 감독/진행 경기/총 득점, 1위 감독, 다음 경기)
- 순위표 (8감독, 엠블럼, 승/무/패/승점)
- 최근 경기 결과
- 득점 순위 Top 5

**미구현**
- 우승 테마 배경 전환
- 득점력/수비력 지표

### 5-2. 리그

**기획**
- **경기 일정** — 경기 목록 표시, 경기 고유 번호(#ID)로 식별
- **순위표** — 승점·승/무/패·득실차
- 경기 결과 입력: 넥슨 경기결과 API 자동 조회 우선, 불가 시 각 감독이 수기 입력

**구현됨 (샘플 UI)**
- `/league/schedule` — 경기 목록, #ID 표시, 결과 입력 폼 UI
- `/league/standings` — 순위표

**미구현**
- API 연동 및 자동 순위 갱신
- 넥슨 경기결과 API 연동

### 5-3. 팀

> **팀 = 감독** — 1감독 1팀

#### 내 팀 정보 (`/teams/my`)

**기획**
- 로그인한 감독 본인의 팀 스쿼드·스킬·포메이션 확인 및 편집
- 포메이션 선택, 선수 배치, 케미 설정, 구단가치/OVR 자동 계산

**구현됨 (샘플 UI)**
- Best 11 포메이션 배치, 감독 스킬 패널

**미구현**
- 팀 편집 UI, API 연동

#### 전체 팀 정보 (`/teams/all`)

**기획**
- 참여 감독 전체의 팀 스쿼드를 **그리드 레이아웃** 으로 표시
- **간략 보기**: 각 감독의 Best 11이 리스트 형태로 표시
- **상세 보기 (팝업)**: 감독이 설정한 포메이션과 선수 배치, 선수 상세 정보

**구현됨 (샘플 UI)**
- 8감독 카드 그리드 (엠블럼, 포메이션, OVR, 승점)
- 감독 선택 시 상세 패널: Best 11 포지션 배치 (`PlayerSlot` 컴포넌트)
- 감독별 **감독 특성** 패널 (랜덤 배정 2~4개, 호버 시 상세 툴팁)
- SVG 방패 엠블럼 (`Emblem.jsx`, 우승 시 ★ 표시)

#### 내 팀 설정

**기획**
- 포메이션 선택 및 선수 배치 구성
- 선수 검색에서 즐겨찾기한 선수 또는 내 선수단에 추가된 선수 목록에서 선택
- **시장가치 자동 합산** — 배치된 선수들의 시세 합계 표시
- **OVR 자동 계산** — Best 11 기준 강화등급 적용 스탯 평균
- **케미 설정 (수동 선택)** — 강화케미/팀케미 효과 수동 선택
- 우측 패널: 적용된 OVR, 케미 설정값 표시

**미구현**
- 내 팀 설정/편집 UI
- 시장가치 합산, 케미 설정, 선수단 편집

## 6. 감독 특성 시스템

8명 감독이 공유하는 **특성 풀**에서 감독마다 2~4개가 랜덤 배정된다. (이름 시드 기반, `managerTraits.js`)

### 6-1. 특성 목록 (7종)

| ID | 이름 | 타입 | 비용 | 설명 |
|----|------|------|------|------|
| `ban-player` | 선수 밴 능력 | 소모형 | 10P | 상대 선수 1명 출전 불가 처리 (경기당 1회) |
| `reduce-squad` | 선수단 감소 능력 | 소모형 | 20P | 상대 선수단 등록 인원 10명으로 제한 |
| `training-coach` | 능숙한 훈련감독 | 패시브 | — | n명(2~5) 선수에게 훈련 코치 부착 |
| `sub-master` | 교체의 명장 | 패시브 | — | 교체 선수 (n BP, 200~500) 규모 운용 |
| `doctor-strange` | 닥터스트레인지 | 소모형 | 10P | 선수 1명 재선발 (경기당 1회) |
| `enhance-master` | 강화의 달인 | 패시브 | — | 강화 성공률 n%(3~8%) 상승 |
| `draw-master` | 뽑기의 달인 | 패시브 | — | n%(8~20%) 확률로 재뽑기 활성화 |

- **소모형** (`oneShot: true`): 사용 시 포인트 소모, 해당 경기 1회 적용
- **패시브** (`oneShot: false`): 지속 효과, 포인트 소모 없음

### 6-2. UI 구현

| 화면 | 경로 | 내용 |
|------|------|------|
| 팀 상세 | `/teams` | 감독 선택 시 특성 패널, 카드 호버 툴팁 (설명 + detail) |
| 특성 투자 | `/traits` | 정복자 포인트식 +/- 투자 (레벨 0~5), 전체 특성 리스트 |

**구현 파일**: `frontend/src/constants/managerTraits.js`, `Teams.jsx`, `TraitsInvestment.jsx`

**미구현**: 특성 사용/소모 게임 로직, 투자 레벨 효과 반영, 서버 저장
