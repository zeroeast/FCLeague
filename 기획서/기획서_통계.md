FC 온라인 친구 리그 전적 사이트의 [통계] 페이지 UI를 설계해줘.

전체 페이지는 "통계" 메뉴 안에 있으며, 상단에 세분화 탭이 2개 있다.

세분화 탭:
1. 팀
2. 선수

탭을 클릭하면 같은 레이아웃 프레임 안에서 내용만 변경된다.
선택된 탭은 활성화 상태를 강조 표시한다.

전체 디자인 방향:
- 친구들끼리 FC 온라인 친선경기 전적을 모아 보는 웹페이지
- 통계/랭킹 중심의 대시보드 UI
- 너무 화려한 게임 UI보다는 깔끔한 스포츠 통계 사이트 느낌
- 카드형 박스 기반 레이아웃
- PC 웹 기준 1200px 내외의 넓은 대시보드
- 각 박스는 흰색 배경, 얇은 테두리, 약한 그림자, 적당한 여백을 가진다
- 데이터가 없을 때도 빈 상태가 어색하지 않게 placeholder를 둔다

공통 레이아웃:
- 상단에 탭 영역 배치
- 탭 아래에는 큰 콘텐츠 영역이 있다
- 콘텐츠 영역 안에는 상단 2개 카드, 하단 3개 카드로 구성한다
- 상단 왼쪽 카드는 가로로 긴 메인 카드
- 상단 오른쪽 카드는 세로형 요약 카드
- 하단에는 동일한 크기의 랭킹 카드 3개를 가로로 배치한다

레이아웃 구조:
[탭 영역]
[메인 대형 카드]                  [우측 요약 카드]

[하단 랭킹 카드 1] [하단 랭킹 카드 2] [하단 랭킹 카드 3]

====================================
팀 탭 화면
====================================

팀 탭은 친구 계정/구단주 단위의 통계를 보여준다.
핵심 목적은 "우리 친구들 중 누가 제일 잘하는가", "누가 공격적이고 수비적인가", "최근 폼이 좋은 사람은 누구인가"를 보여주는 것이다.

팀 탭 구성:

1. 상단 왼쪽 메인 카드
제목: 리그 종합 순위

내용:
- 전체 전적 순위표
- 컬럼: 순위, 구단주, 경기 수, 승, 무, 패, 승률, 득점, 실점, 골득실, 최근 5경기
- 1~3위는 시각적으로 강조
- 최근 5경기는 W/D/L 배지로 표시

2. 상단 오른쪽 요약 카드
제목: 현재 1위 팀

내용:
- 현재 랭킹 1위 구단주의 요약 정보
- 구단주명
- 승률
- 총 경기 수
- 골득실
- 최근 폼
- 간단한 코멘트 예: "최근 5경기 4승 1무"

3. 하단 왼쪽 카드
제목: 팀 총 득점 순위

내용:
- 구단주별 총 득점 랭킹
- 순위, 구단주명, 총 득점, 경기당 득점
- 공격력이 좋은 팀을 보여주는 영역

4. 하단 중앙 카드
제목: 팀 골득실 순위

내용:
- 구단주별 골득실 랭킹
- 순위, 구단주명, 득점, 실점, 골득실
- 전체적인 안정성을 보여주는 영역

5. 하단 오른쪽 카드
제목: 최소 실점 순위

내용:
- 구단주별 최소 실점 랭킹
- 순위, 구단주명, 총 실점, 경기당 실점
- 수비력이 좋은 친구를 보여주는 영역

====================================
선수 탭 화면
====================================

선수 탭은 실제 경기에서 사용한 선수 카드 단위의 통계를 보여준다.
핵심 목적은 "어떤 선수가 제일 잘했는가", "누가 득점/도움/수비에서 활약했는가", "기대보다 부진한 선수는 누구인가"를 보여주는 것이다.

선수 탭 구성:

1. 상단 왼쪽 메인 카드
제목: 현재 선수 파워랭킹

내용:
- 전체 선수 카드 기준 종합 랭킹 TOP 10
- 컬럼: 순위, 선수명, 시즌, 포지션, 출전 수, 골, 도움, 파워점수
- 파워점수는 골, 도움, 출전 수, 경기당 기여도를 종합한 값
- 1~3위 선수는 강조 표시
- 선수 이미지를 작게 표시할 수 있으면 좋다

2. 상단 오른쪽 요약 카드
제목: 워스트 퍼포먼스

내용:
- 기대 대비 가장 부진한 선수 카드 1명 또는 TOP 3
- 친구들끼리 보는 사이트이므로 약간 유머러스한 톤 가능
- 단, 너무 공격적인 표현보다는 "워스트 퍼포먼스", "아쉬운 선수", "먹튀 랭킹" 같은 표현 사용 가능
- 표시 정보: 선수명, 시즌, 출전 수, 골, 도움, 부진 사유 요약

3. 하단 왼쪽 카드
제목: 득점 순위

내용:
- 선수별 총 득점 랭킹
- 순위, 선수명, 시즌, 포지션, 총 득점, 경기당 득점
- 공격수 아이콘 또는 골 아이콘 사용 가능

4. 하단 중앙 카드
제목: 어시스트 순위

내용:
- 선수별 도움 랭킹
- 순위, 선수명, 시즌, 포지션, 총 도움, 경기당 도움
- 플레이메이커 역할이 드러나게 표현

5. 하단 오른쪽 카드
제목: 수비 순위

내용:
- 수비 기여도 랭킹
- 순위, 선수명, 시즌, 포지션, 수비점수
- 수비점수는 태클, 차단, 클리어, 수비 포지션 기여도 등을 기반으로 계산된 값으로 가정
- 수비수와 골키퍼가 돋보이도록 구성

====================================
상호작용
====================================

- 사용자가 "팀" 탭을 누르면 팀 통계 화면이 표시된다.
- 사용자가 "선수" 탭을 누르면 선수 통계 화면이 표시된다.
- 각 카드에는 우측 상단에 기간 필터나 정렬 기준을 둘 수 있다.
- 예: 전체 기간 / 최근 10경기 / 이번 달
- 랭킹 항목을 클릭하면 해당 유저 또는 선수의 상세 통계 페이지로 이동할 수 있게 설계한다.
- 데이터가 없는 경우 "아직 수집된 경기 데이터가 없습니다" 같은 빈 상태 메시지를 표시한다.

====================================
디자인 톤
====================================


- 포인트 컬러는 파란색 또는 FC 온라인 느낌의 짙은 네이비/골드 계열
- 숫자 통계는 굵고 명확하게 표시
- 순위는 배지 형태로 표시
- 1위는 금색, 2위는 은색, 3위는 동색 느낌으로 강조
- 모바일보다는 PC 웹 대시보드 우선
- 전체적으로 깔끔하고 직관적인 스포츠 랭킹 대시보드 느낌
